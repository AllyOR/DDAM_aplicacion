ENTRENAMIENTO CANINO

MUERTO

Este truco puede ayudarte a mantener tranquilo a tu perro.

  • Primero debes sostener recompensa en tus manos.
  • Debes guiarlo mediante las recompensas para que siga el movimiento de tus manos.
  • Cuando se encuentre junto a tu mano en el suelo di "HAZTE EL MUERTO".
  • Trata de mantenerlo totalmente recostado mientras le das la recompensa.
  • Ahora aleja cada vez más la mano y repite el truco.
  • Recompénsalo cada vez que se queda quieto un tiempo.

Este truco require de varias repeticiones.
